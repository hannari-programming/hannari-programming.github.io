## 多言語対応のルール

### UIテキストの追加・変更

`src/i18n/index.ts` の `ui` オブジェクトに **3言語セットで** 追加する。
1言語だけ追加は禁止。`zh` が未翻訳の場合は日本語テキストをそのまま入れてプレースホルダーコメントを付ける。

```typescript
// src/i18n/index.ts
export const ui = {
  ja: { 'new.key': '新しいテキスト' },
  en: { 'new.key': 'New text' },
  zh: { 'new.key': '新しいテキスト' }, // TODO: zh translation needed
};
```

### 新しいページの追加

3言語分のファイルを同時に作成する:

```
src/pages/ja/about.astro   ← メインコンテンツ
src/pages/en/about.astro   ← 英語版
src/pages/zh/about.astro   ← 中国語版（コンテンツ未翻訳なら募集バナーを表示）
```

中国語版のコンテンツが未翻訳の場合は `zh/index.astro` の募集バナーパターンを流用すること。

### 翻訳ヘルパーの使い方

```astro
---
import { getLangFromUrl, useTranslations } from '../../i18n';
const lang = getLangFromUrl(Astro.url);
const t    = useTranslations(lang);
---
<h1>{t('nav.events')}</h1>
```

ハードコードされた日本語文字列をページ内に直接書かない。必ず `t()` を通す。
